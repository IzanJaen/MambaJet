# 🎨 Paleta de Colores — MambaJet

La paleta de colores de la aplicación sigue el sistema **Material 3** y una estética "Cyber-Clean". Se han elegido colores para transmitir eficiencia, tecnología y exclusividad (estilo *Elite Member*), manteniendo un contraste impecable para la accesibilidad tanto en el modo claro como en el modo oscuro.
