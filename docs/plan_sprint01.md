# Sprint 01 – Planning Document

## 1. Sprint Goal
El objetivo principal del Sprint 01 es crear el esqueleto de una aplicacion "Travel Planner" llamada MambaJet, donde hacemos el setup tanto del proyecto como del GitHub e implementamos el Core de la aplicación y sus principales pantallas, con su navegación y customización.
