# 🐍 MambaJet
App to plan your travel at a Black Mamba's speed

## 📖 Sobre el Proyecto
MambaJet es mucho más que un planificador de viajes convencional; es un ecosistema diseñado para la élite viajera. Construida bajo una rigurosa estética "Cyber-Clean", la aplicación permite a los usuarios gestionar sus viajes como verdaderas "Misiones". 

Desde el control milimétrico del presupuesto hasta la organización cronológica de actividades (Waypoints) y el almacenamiento de recuerdos en una bóveda fotográfica exclusiva. Todo ello garantizando una experiencia de usuario ultra fluida, modular y con una arquitectura base preparada para futuras integraciones avanzadas de Inteligencia Artificial y geolocalización.

---
## 🔥 Características Principales
* **Itinerarios Dinámicos (Timeline):** Organiza vuelos, hoteles, restaurantes y exploraciones en una línea de tiempo clara, minimalista y fácil de leer.
* **Control Financiero Avanzado:** Monitoriza tu presupuesto total y el gasto actual en tiempo real mediante tarjetas de resumen y barras de progreso intuitivas.
* **Mamba Intelligence (Ready):** Estructura base implementada para un futuro motor de IA capaz de analizar gastos, sugerir optimizaciones de presupuesto y generar rutas inteligentes.
* **Radar Táctico (Mapas):** Módulo visual proyectado para trazar actividades y descubrir Puntos de Interés (POIs) como hoteles o gastronomía cercana.
* **Bóveda de Capturas:** Galería inteligente integrada para almacenar, filtrar y organizar los recuerdos fotográficos asociados a cada misión específica.
* **Modo Oscuro Dinámico:** Interfaz adaptable al 100% que responde a las preferencias del sistema o del usuario, utilizando la paleta de colores de Material 3.
* **Gestión de Perfil Élite:** Panel de ajustes centralizado para gestionar la cuenta, preferencias de notificaciones, selector de idioma y configuraciones del dispositivo.

---
## 📲 Tecnologías Usadas
* **Lenguaje Core:** Kotlin
* **UI Toolkit:** Jetpack Compose (100% Declarativo)
* **Sistema de Diseño:** Material Design 3 (Soporte nativo Light/Dark Mode)
* **Arquitectura:** MVVM (Model-View-ViewModel) enfocada en la escalabilidad y separación de responsabilidades (SRP).
* **Enrutamiento:** Jetpack Navigation Compose (Single-Activity Architecture)
* **Control de Versiones:** Git & GitHub
